# 🦆 Duck & Cover

Objects fly at you. Physically duck below your webcam to dodge. Pop up and yell "BANG" to shoot back. Prairie dog with a death wish energy. Combo system rewards dodge-dodge-shoot streaks.

## ▶️ [Play Now](https://ashleywolf.github.io/duck-and-cover/)

## How to Play

Open in Chrome, allow webcam + microphone. Stand in front of your camera.

- **Duck** below the frame to dodge projectiles
- **Stand up and yell** to shoot
- Rocks must be dodged, targets can be shot, bombs MUST be shot

## Tech

MediaPipe PoseLandmarker (nose Y tracking for duck detection) + Web Audio API (volume spike for shout detection). HTML5 Canvas. Single file, CDN-loaded.

## Input

Webcam (full body pose) + Microphone (volume detection)
